$.parse

A super light-weight AJAX wrapper for Parse.com's wonderful database service.

  1. parse.sh
    Generate your basic Auth Key from Terminal
  2. demo.js
    Paste auth key on line 11 -> "auth : [your_auth_key]"
  3. Play!

NOTE :

  • You need to serve this page from the file:// protocol or use this page in a Chrome extension because of Cross Domain Security limitations.
  • Remember that you are publicly displaying your basic auth key in demo.js
  • Tip! Adding a server proxy for $.parse to parse.com wouldn't be too hard if you want to hide your auth key and add some user restrictions.
